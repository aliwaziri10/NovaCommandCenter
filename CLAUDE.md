# Nova Command Center Bootstrap

## Startup Procedure

1. Read `brain/INDEX.md`.
2. Read every file listed there in order.
3. Inspect the current repository:
   - backend/
   - frontend/
   - .github/
4. Compare documentation with the actual code.
5. If documentation is outdated, update it.
6. Continue the highest-priority unfinished task automatically.
7. Before ending the session:
   - Update PROJECT_STATE.md
   - Update TASK_QUEUE.md
   - Record important decisions in DECISIONS.md
   - Record new bugs in KNOWN_BUGS.md
   - Append work completed to SESSION_LOG.md

Never rely on chat history.

The repository is the single source of truth.
